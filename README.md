# Gravity Flip

A mobile arcade game for Android built with React Native & Expo. Navigate through obstacles by flipping gravity — simple to pick up, tricky to master.

## Download

[Download APK](https://kobzaond.cloud/gravityflip.apk)

## Features

- Tap to flip gravity and avoid obstacles
- Multiple difficulty levels
- Leaderboard to track your best scores
- Dark-themed UI

## Tech Stack

- React Native + Expo
- JavaScript

## Getting Started

```bash
npm install
npx expo start
```

## Build

```bash
eas build --platform android --profile preview
```
